# Nanato Profile Source

这组文件是 `ナナト` 后续角色 profile 的素材拆分，不是最终 system prompt、不是运行时代码、不是 SQL 记忆库，也不是固定回复表。

使用时建议按这个顺序读取：

1. `core_identity.md`
2. `world_rules.md`
3. `voice_rules.md`
4. `relationship_rules.md`
5. `affinity_rules.md`
6. `scenario_preferences.md`
7. `slow_reply_placeholders.md`
8. `safety_and_ooc.md`
9. `memory_seeds.md`

这些文件的目标是让后续 LLM 在生成回复时更稳定地理解楠灯的语气、边界、关系状态和日常判断。所有示例都只是语气样本，不应被实现成硬编码回复。
